Visão do Projeto

Poison Linux é uma distribuição GNU/Linux 64 bits, com foco em computação para desktops mais (modestos)
A base utilizada é o Slackware™ Linux com inspirações do Absolute Linux e Salix.

  • Pacotes 100% compatíveis com Slackware
  • sbopkg incluso com total compatibilidade com SlackBuilds
  • Usa o slapt-get como gerenciador, com resolução de dependências
  • Foco em window manager i3wm ao invés de algum ambiente desktop completo.
  • Várias automações para tornar o sistema mais mastigado para utilização
  • Mais compacto que o Slackware.
  • Repositório de pacotes próprios.
  • Segue a filosofia KISS.
